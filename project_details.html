1 Project Overview

What is the EMS App?

The Event Management System (EMS) is a full-stack web application that allows organizations to plan, manage, and track events seamlessly. It includes tools for task assignment, contact management, event registration, venue tracking, and Kanban-style task boards.

Key Features:

  • Event creation and scheduling
  • Pre-event task manager with drag-and-drop Kanban board
  • Contact management and automatic CRM creation on registration
  • Venue facilities listing and assignment

What’s Next:

  • Add role-based permissions for admins, organizers, and attendees
  • Integrate ticket generation for event registrants
  • Build event analytics dashboard and reporting tools

2 The Challenge

Existing AR applications suffered from cluttered interfaces and overwhelming options. Users reported frustration with unintuitive controls and excessive information density.

Our research identified three key pain points:

  • Overwhelming UI with too many options visible at once
  • Inconsistent AR tracking in urban environments
  • Lack of contextual information hierarchy

3 Our Solution

Context-Aware Interface

Dynamic UI that adapts to the user's environment and activity, showing only relevant controls.

Smart Landmark Recognition

Proprietary algorithm that identifies architectural features with 92% accuracy in diverse lighting conditions.

Information Layering

Three-tiered content system that allows users to drill down from glanceable info to detailed historical data.
