Technologies Used

  • Laravel (PHP) – Backend framework
  • Laravel Breeze – Authentication and frontend starter
  • MySQL – Database for storing events, tasks, contacts, etc.
  • React – Frontend JavaScript library for building UI
  • Inertia.js – Bridges Laravel backend with React frontend
  • Tailwind CSS – Utility-first CSS framework for styling
  • Heroicons – SVG icons used across the UI
  • @dnd-kit – Drag-and-drop Kanban-style task board
  • Framer Motion – Animation library for React
  • Vite – Fast development and build tool
  • Axios – HTTP requests between frontend and backend
